Answers of four Important JavaScript Questions


  1. 1. What is DOM and what is the purpose of DOM?

    2. Ans: The Document Object Model (DOM) is a way for web documents to talk to programs. It shows what the page looks like so that programs can change the structure, style, and content of the document. The goal of the DOM is to give developers a standard way to access and change the elements of a web page. This lets developers make web pages that are dynamic and interactive.


  2. 2. How will you select HTML elements using DOM? Name the DOM methods.

    3. Ans: DOM can be selected in many ways. There are few common ways I have shown to select dom :

    • 1. getElementById(): It picks an element based on its ID attribute.
    • 2. getElementByClassName(): It picks out all of the elements that have the given class name.
    • 3. getElementByTagName(): It picks out all the elements that have the given tag name.
    • 4. querySelector(): It takes a CSS selector and finds the first element that matches it.
    • 5. querySelectorAll(): It picks out all of the elements that match a CSS selector.


  3. 3. What is event bubble?

    4. Ans: Event bubbling is when events move up the DOM tree from child elements to their parents. When an event happens on a child element, it is first handled by the child element, then sent up the tree to its parents, and so on. This can be useful for making event handlers that work on multiple elements at once, but if it's not done right, it can also cause unexpected behavior.


  4. 4. What is a callback function and why will you use it?

    5. Ans: Callback functions are provided to other functions and executed afterwards. Event-driven programming uses callback functions to define bespoke behavior based on user actions or other events. They can also manage asynchronous processes like retrieving server data. By giving a callback function as a parameter, the program can continue executing while the operation is in progress and execute the callback function thereafter.
